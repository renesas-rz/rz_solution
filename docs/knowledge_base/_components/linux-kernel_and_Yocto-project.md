## Linux Kernel and Yocto Project

This section provides links to official resources for the Linux kernel and the Yocto Project, which form the foundation of RZ MPU Linux solutions.

### Linux Kernel

* [The Linux Kernel Archives](https://www.kernel.org/){: target=_blank } — Official archive site for Linux kernel source code and releases.
* [CIVIL INFRASTRUCTURE PLATFORM (CIP)](https://cip-project.org/){: target=_blank } — Industrial-grade Linux initiative that provides long-term maintained Linux kernels for embedded and industrial systems.
For detailed information on the CIP project, including supported kernel versions and maintenance periods, refer to the [CIP Wiki](https://wiki.linuxfoundation.org/civilinfrastructureplatform/start){: target=_blank }.
